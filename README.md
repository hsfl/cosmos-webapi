# cosmos-webapi

## Requirements

- Node.js

## Installation

```bash
# Clone repository
git clone git@github.com:hsfl/cosmos-webapi.git

# cd into repository
cd cosmos-webapi

# Install dependencies
npm i
```

## Running

```bash
npm run dev
```

## Unit Tests
```bash
mocha
```
